## Installation

1. Open **VS Code**
2. Press `Ctrl+P`
3. Run: `ext install saqibbedar.bedarx-pro`
4. Press `Ctrl+K Ctrl+T`
5. Select your preferred theme:
   - BXP Dark Default
   - BXP Midnight Aurora
   - BXP Stellar Nova
   - BXP Crystal Light

## Requirements

- VS Code v1.84.0 or newer
